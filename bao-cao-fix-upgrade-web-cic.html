Bước 1: Backup toàn bộ

Tạo cic-code-backup-20260303.zip (593 MB) trước khi can thiệp. Lưu bản gốc từng file vào thư mục backup riêng.

Bước 2: Rà soát Database (Spam Scan)

Viết script quét toàn bộ database tìm spam keywords (casino, gambling, cá cược...), ký tự tiếng Nhật, URL đáng ngờ, và bản ghi bất thường trong 30 ngày.

Bước 3: Quét mã độc trên file hệ thống

Tìm eval(), base64_decode() trong source code. Kiểm tra .htaccess chống inject redirect. Phát hiện file hash-named bất thường.

Bước 4: Vá lỗ hổng bảo mật

Fix SQL Injection (4 file), xóa get_magic_quotes_gpc(), sửa constructor exploit, chuyển create_function() sang closure.

Bước 5: Cập nhật .htaccess bảo mật

Thêm redirect cic.com.vn → www.cic.com.vn. Cập nhật URL pattern sản phẩm. Thêm no-cache cho HTML/JS/CSS.
